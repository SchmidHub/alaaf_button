### ESP32 Source for random Sounds to Bluetooth Speaker

Use an ESP32 as a source device to play a randomly chosen .wav-sound effect, when a specific button has been pressed.

# Remarks
The code is optimized to work with PCM WAV 8-Bit mono files.

# Used resources
[Library used for bluetooth connection](https://github.com/pschatzmann/ESP32-A2DP)
[Filesystem used for locally storing .wav files](https://github.com/earlephilhower/arduino-littlefs-upload)
